npm install

Install fping. Installation depends on your linux distribution.

>> http://fping.org

Create a config file using etc/config.json as a template, with the collector name and ApiKey

{
"collector": {
  "name": "DC2 Internal"
},
"numCPUs": 1,
"serverUrl": "https://app.raintank.io",
"apiKey": "[Your API KEY]",
"pintServerPort": 8080
}

Start the app

nodejs app.js -c /path/to/config.json
